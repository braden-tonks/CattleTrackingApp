# Cattle Tracking Application

A full-stack livestock management Android app designed to help small farmers easily track and manage their cattle.  
Built using **Kotlin**, **Jetpack Compose**, and **Supabase**, this project streamlines recordkeeping, weight monitoring, and tag management on an android mobile device.

---

## Overview

The Cattle Tracking App was developed as part of our Capstone Project at Missouri State University (2025).  
Our goal is to provide small-scale farmers with a simple, modern solution for tracking livestock data, reducing manual paperwork, and improving herd management efficiency.
We also wanted to develop our skills and challenge ourselves in full stack development, also learning how to implement outside tools like NFC scanning into our app.

---

## Features

1. Register and track cattle with unique tag numbers  
2. View and manage detailed cow profiles (birth date, sex, lineage, etc.)
3. Record vaccination and health history
4. Track calf weights and growth over time
5. NFC support for scanning ear tags
6. Supabase integration for real-time data sync
7. Clean, intuitive UI built with Jetpack Compose
